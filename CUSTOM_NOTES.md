## Custom Changes

This workspace contains the following customizations on top of upstream:

1. Controls editor button icon enhancements
   - add local custom icon import
   - add custom icon size adjustment
   - add custom icon opacity adjustment
   - add custom pressed color adjustment

2. Icon pack manager for controls editor
   - add icon pack import
   - add icon pack export
   - add icon pack selection/removal
   - support InputBridge-compatible `.ipk` icon packs
   - improve icon pack import compatibility and error reporting

3. Trackpad editor enhancements
   - add "mouse button while moving" option for `TRACKPAD`
   - mouse press landing point defaults to screen center
   - add `X` center offset adjustment
   - add `Y` center offset adjustment
